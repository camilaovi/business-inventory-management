# Business Inventory Management

Full stack web application for managing products, inventory, customers, and sales for a small business.

## Project objective

The objective of this project is to build a functional web system that allows a small business to manage products, inventory, customers, and sales from a single platform.

This project is part of my software engineering portfolio and is designed to strengthen skills in frontend development, backend development, databases, API design, version control, and deployment.

## Main features

- Product management
- Inventory control
- Customer registration
- Sales registration
- Low stock alerts
- Basic business reports
- User authentication
- Dashboard with business indicators

## Technologies planned

- React
- Node.js
- Express
- PostgreSQL
- Git and GitHub
- Postman
- Vercel
- Render
